# Tash

File Explorer With tag system

Built with Tauri v2, React, TypeScript, Mantine UI, and Rust.

## Features

- Browse files and folders with list or grid views
- Tag-based file organization with SQLite storage
- Image thumbnails via Freedesktop thumbnail cache
- Full-text search with Tantivy
- Copy, move, rename, delete file operations
- Archive extraction (.zip, .7z, .rar, .tar.*)
- Virtual grid rendering for directories with thousands of images
- Theme customization

## Development

```bash
npm install
npm run tauri dev
```
