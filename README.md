# Tash

File Explorer With tag system

Built with Tauri v2, React, TypeScript, Mantine UI, and Rust.

## Few words from author


Core function is "tag" system, you can assing tag to every file or directory and then browse files via tags instead of normal way. There is internal image viewer, so far no option to turn it off but there is way to open files normaly. Also if file can be market as exectuable (like shell scripts) then double click will open them in terminal. 
I tried, i tried to make it work with large directoires of photos, but everything over 2k files when miniatures are active is freezing. 1k is fine tho and if you turn off miniatures even 7k files have slight delay.
